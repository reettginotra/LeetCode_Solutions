#  LeetCode Solutions

A curated collection of my **LeetCode** solutions written in **C++ & python**, organized by topic and updated regularly.

The goal of this repository is to:
-  Document my problem-solving journey.
-  Help students understand solutions for problems that may not have clear explanations online.
-  Maintain consistency through daily practice.
-  Prepare for coding interviews while improving algorithmic thinking.




##  Daily Challenge

The `Daily_Challenge` folder contains my solutions to LeetCode's Daily Challenge.

I aim to solve at least **one problem every day** to strengthen my Data Structures and Algorithms skills while maintaining consistency.
